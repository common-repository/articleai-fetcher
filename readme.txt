== Description ==

This plugin fetches and posts articles from ArticleAI.io to your WordPress site. The plugin fetches content for each category from different queues and posts them automatically. Queues must be configured for each category and their frequency of content fetching can also be customized. 

The plugin is simple to configure and use:

1. Go to ArticleAI
2. Specify the API Key.
3. Select the queue name for each category. Queue names are set in ArticleAI.io when articles are created.
4. Choose the frequency of fetching new content.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/articleai` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to ArticleAI to configure the plugin.

== Frequently Asked Questions ==

= What is the queue name? =

The queue name is set in ArticleAI.io when you create articles. It is used to fetch articles from the corresponding queue.
